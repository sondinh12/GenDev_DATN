1. Migration - dùng để tạo bảng trong cơ sở dữ liệu 
- Thực hiện migration bằng lệnh : php artisan migrate
- Tạo migration cho bảng mới: php artisan make:migration create_ten_bang_table
- để rollback lại câu lệnh vừa thực hiện: php artisan migrate:rollback
- Để rollback lại nhiều câu lệnh: php artisan migrate:rollback --step=**
- Để rollback lại toàn bộ cơ sở dữ liệu sau đó migration  lại từ đầu(reset db)
    php artisan migrate:refresh

- tao model: php artisan make:model TenModel
2. Seeder
- Để tạo file seeder: php artisan make:seeder TenfileSeeder
3. Factory
- Để tạo file Factory: php artisan make:factory TenfileFactory
